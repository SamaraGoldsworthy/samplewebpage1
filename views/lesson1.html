These are the Lesson 1 Labs

Syntax

I learned what syntax is, how to use parentheses, curly braces and semicolons.

Placeholder

Next I learned how to use placeholders. How to use printf, and special characters such as /n.

Terminal

I learned how to use the command-line interface and the commands involved in the terminal to navigate the workspace. I also learned how to create, move and rename folders and files.

Data Types

In this lab, I learned about the common data types used in CS50 and the data input functions for different data types.

Variables

In this lab I learned the difference between variables in math and CS, assigning a value to, creating and naming and getting user input for a variable.

Operators

In this lab I learned how math operators work in C, how and when to use the remainder operator, and the different ways of using assignment operators.

Booleans (Expressions/Conditionals)

In this lab I learned how to use Boolean expressions in C, why and how to combine Boolean operators and why conditional statements are so important.

While Loops

In this lab I learned what a while loop is, and how and when to use them.

Do While Loops

In this lab I learned what the difference between a do while and while loop is, and how and when to use them.

For Loops

In this lab I learned why we have a for loop and how to use a for loop.

Magic Numbers

In this lab I learned what is meant by magic numbers and why magic numbers are considered poor design.
